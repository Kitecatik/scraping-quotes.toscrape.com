# scraping-quotes.toscrape.com
In this project, I demonstrated the skills of parsing static website. Also received data has been added to the table.

# Installation instructions
Firsty, download the repository with the command:
```
git clone https://github.com/Kitecatik/scraping-quotes.toscrape.com.git
```
Then, then go to the root directory of the project and install the libraries with the command:
```
pip3 install -r requirements.txt
```
Next then run the script with the command:
```
python3 main.py
```
